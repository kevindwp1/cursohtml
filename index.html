Etiquetas semanticas estructurales

DIV-es una etiqueta contenedora que no tiene semantica
HEADER-Representa la cabecera de un sitio o una seccion
MAIN-Define la seccion principal del documento
FOOTER-representa el pie de pagina de un sitio web o de una seccion del sitio
NAV-Representa un conjunto de enlaces o botonera
ARTICLE-Representa un a seccion autocontenido
ASIDE-Representa un contenido complementario o secundario
SECTION-Representa una seccion de contenido generico
ADDRES-representa una direccion fisica o informacion de contacto

Meta etiquetas para SEO


     
CONICA

            La etiqueta canonica le dice a los motores de busqueda que la pagina colocada en la etiqueta
            es la pagina oficial o la pagina original 

                  <link rel="canonica" href="https://tudominio.com/la-url-canonica">
     
   
    
ICON

            En la etiqueta icon se coloca el icono que va en la parte superior y su formato debe de ser .ico o png

                <link rel="icon" href="img/g.png">
    
Apple-touch-icon
  
             Esta etiqueta funciona al momento de descargarse la URL en el dispositivo movil

                <link rel="apple-touch-icon" href="img/g.png">

    
Cambiar el color de la barra de direcciones

              Esta etiqueta funciona para cambiar el color de la barra de direcciones del dispositivo

                <meta name="theme-color" content="#ff6600">
